# FEATURES.md

## Table Stakes (Must Have)
- **MetaMask Connectivity**: Secure authentication and transaction signing.
- **Admin Dashboard**: Update creator shares, adding/removing recipients.
- **Creator Dashboard**: Read-only view of split percentages and accumulated earnings.
- **Revenue Overview**: Historical transaction tracking grouped by hash.

## Differentiators
- **One-Click Distribution**: Direct UI trigger for 0xSplits distribution functions.
- **Real-Time Rights Tracker**: Visualization of ownership percentages.
- **In-App Demo**: One-click simulated environment for testing flows.
