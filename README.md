# LUNIM: Creative Rights & Revenue Tracker

LUNIM is a premium, cinematic-grade Web3 platform designed for creative rights management and revenue distribution. Built for modern content businesses like **HBO**, it enables transparent royalty splitting, real-time payment tracking, and smart-contract-backed financial operations.

## 🚀 Key Features
- **Gasless Onboarding**: Sign up with Google/Email; no MetaMask or ETH required.
- **Account Abstraction (ERC-4337)**: Automated smart account infrastructure using Safe.
- **Sponsored Transactions**: 100% gas-free revenue distribution via Alchemy Paymaster.
- **Cinematic Dashboard**: Premium HBO-themed UI with advanced analytics.
- **Financial Transparency**: Accurate tracking of payments in cents with USD reporting.

## 📦 Tech Stack
- **Frontend**: Next.js 14, React, Tailwind CSS, Framer Motion.
- **Web3 Layer**: Privy (Auth/Wallets), Safe (Smart Accounts), Viem/Wagmi.
- **Infrastructure**: Alchemy (Bundler/Paymaster), Base Network.
- **Database**: Supabase.

## 🛠️ Local Development
1. `npm install`
2. Configure `.env.local` with Privy and Alchemy keys.
3. `npm run dev`

## ⛓️ Blockchain Operations
This platform operates on **Base Sepolia** (Testing) and **Base Mainnet** (Production). Smart accounts are automatically initialized upon user login.
